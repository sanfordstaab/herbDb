How this database works This database consists of a web of pages that can be referenced by various categorized lists and which cross-link to each other. This home page is where you can browse any of these pages via various lists which you can then follow to learn whatever you like.
Articles Articles are pages mostly about a specific herb but can also contain pages on any herb-related subject.
 
Properties Properties are things which herbs posess. An herb may have many properties and many herbs may affect the same property.
